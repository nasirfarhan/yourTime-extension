# yourTime-extension
A chrome extension buddy that displays your reading time whenever you read any online documentation
This is my another practice project just to learning how to build extensions
Instructions to use:
 Step 0) Clone the repo in your local device
 Step 1) Go to chrome://extensions
 Step 2) Enable developer mode
 Step 3) Click on Load Unpacked 
 Step 4) Select the repo 
 Step 5) Enable the extension
 Step 6) Go to wiki , medium or similar websites
 Step 7) See the reading time just below the heading!!!!!
